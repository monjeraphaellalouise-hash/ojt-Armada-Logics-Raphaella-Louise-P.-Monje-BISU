Project Name Description Year
Gas Leakage Detector with SMS Alert using GSM Module and Arduino This project detects gas leaks using a gas sensor connected to an Arduino. When gas is detected, the system sends an alert message to the user via WhatsApp, allowing quick action to prevent accidents and improve safety. 2024
Traffic Sign Recognition using Convolutional Neural Networks (CNN) This app uses Convolutional Neural Networks (CNN) to automatically recognize traffic signs from images captured by a camera. The system analyzes visual features in real time and identifies traffic signs to help improve road safety and driver awareness. 2024
Art on the Wall Is an Arduino-based drawing system that creates artwork on a vertical surface using motors and a pen. The system uses stepper motors, strings, and a servo motor to control the movement and position of the pen. By programming the Arduino, different patterns and designs can be drawn automatically, combining creativity with automation for educational and artistic purposes. 2025
